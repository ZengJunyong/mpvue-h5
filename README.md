# mpvue-h5

> weapp and h5 project. 一套代码，同时适配微信小程序和Vue H5应用。

开发过程中，使用的Vue语法尽量让两者都支持，即mpvue不支持的某些Vue语法，就不用。

## Build Setup

``` bash
# install dependencies
cd mpvue-h5
npm install

# Vue H5 at localhost:8080
// H5开发运行
npm run dev:h5
// H5打包
npm run build:h5

# weapp
// 小程序开发运行
npm run dev:weapp
// 小程序打包
npm run build:weapp
```
